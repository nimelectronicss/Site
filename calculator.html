📟 Simple Calculator in C

📖 About The Project

این یه ماشین حساب خط فرمان ساده است که با زبان C نوشته شده. کاربر دو عدد و یک عملگر (+, -, *, /) رو وارد می‌کنه و برنامه نتیجه رو محاسبه می‌کنه.

این پروژه اولین قدم من در یادگیری برنامه‌نویسی C بود و مفاهیم پایه‌ای مثل متغیرها، شرط‌ها (switch-case)، ورودی/خروجی و حلقه رو پوشش می‌ده.

💻 Full Source Code

#include <stdio.h>

int main() {
    char op;
    double num1, num2, result;

    printf("Enter operator (+, -, *, /): ");
    scanf("%c", &op);
    printf("Enter two numbers: ");
    scanf("%lf %lf", &num1, &num2);

    switch(op) {
        case '+':
            result = num1 + num2;
            break;
        case '-':
            result = num1 - num2;
            break;
        case '*':
            result = num1 * num2;
            break;
        case '/':
            if(num2 != 0)
                result = num1 / num2;
            else {
                printf("Error: Division by zero!\n");
                return 1;
            }
            break;
        default:
            printf("Invalid operator!\n");
            return 1;
    }
    printf("Result: %.2lf\n", result);
    return 0;
}

🔧 Key Concepts Learned

  • متغیرها و نوع داده (double, char)
  • گرفتن ورودی از کاربر با scanf
  • ساختار شرطی switch-case
  • مدیریت خطا (مثل تقسیم بر صفر)
  • خروجی با printf
